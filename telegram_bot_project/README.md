# Telegram Bot: Питание и тренировки

## Установка

Создайте виртуальное окружение:

```bash
python -m venv venv
```

Активируйте окружение:

- **Windows**:
  ```
  venv\Scripts\activate
  ```

- **Mac/Linux**:
  ```
  source venv/bin/activate
  ```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python bot/main.py
```
